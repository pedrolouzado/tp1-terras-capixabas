    Sobre

    Olá! Meu nome é Pedro Marcos e este projeto foi desenvolvido para apresentar algumas das principais belezas turísticas do Espírito Santo. O site utiliza HTML, CSS, JavaScript, JSON e Bootstrap para criar uma experiência dinâmica e responsiva.

    O objetivo do projeto é divulgar destinos incríveis do estado, destacando praias, montanhas, turismo religioso e ecoturismo.
